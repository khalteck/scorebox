## Build info

- Built with React.js & Tailwind
- ContextApi for state management
- Hosted with Netlify
- Local JSON files are used as dummy data for data driven components

## Responsiveness

- The webapp is responsive to all screen sizes including mobile & Tabloids..

- Toggle Right sidebar + transitions and abimations
